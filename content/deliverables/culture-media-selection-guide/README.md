# Culture Media Selection Guide — Start Here

A practical guide to choosing, qualifying, and controlling microbiological media
for QC testing — so every plate is traceable to a qualified, fit-for-purpose lot.

## What's inside

1. **Culture Media Selection Guide (PDF)** — how to match media to the method,
   common media by purpose, growth promotion testing, preparation & sterilization,
   equivalence/substitution, and storage & expiry.
2. **Media & GPT Log (CSV/Excel)** — a record template for media lot, prep, pH,
   growth-promotion strains, recovery and inhibition results.

## How to use it

1. Use the guide when setting up a new test, qualifying a media lot, or preparing
   for an audit question on media control.
2. Record each lot and its growth-promotion result in the log before release.
3. Pair this with the **Culture Media Selection** workflow for the step-by-step
   method.

## Important

General educational workflow. Verify against the applicable pharmacopeia
(USP/EP/JP), your approved SOPs, and your validated quality system. Not a
substitute for QA approval or a validated method.
