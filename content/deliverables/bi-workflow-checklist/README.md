# BI Workflow Checklist — Start Here

A practical checklist for using biological indicators (BIs) to challenge and
confirm a sterilization or depyrogenation process — so a release decision rests
on both the physical cycle data and a valid biological challenge.

## What's inside

1. **BI Workflow Checklist (PDF)** — six sections: BI selection, placement,
   exposure & recovery, incubation & reading, interpretation, and
   investigation & release.
2. **BI Run Log (CSV/Excel)** — a template to record each BI lot, organism,
   D-value, population, location, control type, and result.

## How to use it

1. Use the checklist when validating a cycle or running routine BI challenges.
2. Record every BI and its controls in the run log — the positive control is
   non-negotiable; without it a "no growth" result proves nothing.
3. Pair this with the **Biological Indicator Workflow** for the step-by-step
   method.

## Important

General educational workflow. Verify against the applicable regulations and
standards, your approved SOPs, and your validated quality system. Not a
substitute for QA approval or a validated method.
