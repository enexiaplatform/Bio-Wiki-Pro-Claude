# OOS Investigation Template

General educational workflow — verify against your approved OOS SOP, applicable
regulations, and validated quality system. Not a substitute for QA approval.

---

- Reported result: ______________________

Retain the original preparation, solutions, and data. Do not discard anything

---

## 2. Phase I — Laboratory Assessment

Assess for an assignable laboratory cause before any retesting. Check each area
and record findings.

- Calculation and transcription verified: Pass / Fail / NA — notes: __________
- Standard and reagent preparation, identity, and expiry: Pass / Fail / NA — notes: __________
- System suitability / instrument performance and calibration: Pass / Fail / NA — notes: __________
- Sample preparation and dilution scheme: Pass / Fail / NA — notes: __________
- Glassware, column, and consumables: Pass / Fail / NA — notes: __________
- Analyst technique reviewed with analyst (no blame, factual): Pass / Fail / NA — notes: __________

**Phase I conclusion:**

- Assignable laboratory cause identified: Yes / No
- If Yes, describe the cause and the objective evidence: ______________________
- Original result invalidated with justification (QA approved): Yes / No

If no assignable cause is found, do **not** invalidate the result — proceed to
Phase II.

---

## 3. Hypothesis Testing (only if scientifically justified)

Retesting or re-preparation is permitted only under a documented, QA-approved
hypothesis — never to "test into compliance".

- Hypothesis: ______________________
- Test plan (who, what, how many): ______________________
- QA pre-approval: Name __________ Date __________
- Outcome and interpretation: ______________________

A passing retest does **not**, on its own, invalidate the original OOS.
