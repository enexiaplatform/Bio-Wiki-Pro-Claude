# CAPA: getting the effectiveness check right

Most CAPA findings aren't about missing actions — they're about **failing to
prove the action worked**.

## What a good effectiveness check needs

1. **Quantitative criteria** — set in advance and measurable (e.g. recurrence = 0 across 3 batches).
2. **A clear time window** — checked over enough runs/batches to be meaningful.
3. **Objective evidence** — data, not "training completed".

## Common mistakes

- Closing the CAPA as soon as the action is done, before checking effectiveness.
- Using "training completed" as the effectiveness evidence.
- Not tying the check back to the original metric that triggered the issue.
