# Analytical Method Transfer

Method transfer is the documented process that qualifies a receiving laboratory
to run a validated method with the same reliability as the originating lab. A
failed transfer stalls product release, so the protocol design matters.

## Transfer approaches

- **Comparative testing** — both labs test the same samples; results are
  compared against predefined acceptance criteria.
- **Co-validation** — the receiving lab participates in the original validation.
- **Revalidation** — the receiving lab repeats validation in full.
- **Transfer waiver** — justified when the method is compendial or the receiving
  lab already has documented equivalent experience.

## Designing the protocol

Define acceptance criteria for accuracy and precision *before* testing, use
representative (ideally aged or stressed) samples, and confirm both labs share
equivalent instruments, reference standards, and analyst training.
