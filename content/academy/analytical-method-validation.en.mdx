# Analytical Method Validation (ICH Q2)

Method validation demonstrates that an analytical procedure is fit for its
intended purpose. ICH Q2(R2) defines the characteristics to evaluate and which
ones apply depending on the test type.

## The core characteristics

- **Specificity** — ability to measure the analyte unequivocally in the presence
  of impurities, degradants, and matrix.
- **Accuracy** — closeness of the measured value to the true value (often via
  recovery studies).
- **Precision** — closeness among repeated measurements: repeatability,
  intermediate precision, and reproducibility.
- **Linearity & range** — proportional response across the working interval.
- **LOD / LOQ** — the smallest amount detectable and reliably quantifiable.

## Which characteristics apply?

| Test type | Typically required |
| --- | --- |
| Identification | Specificity |
| Impurities (quantitative) | Specificity, accuracy, precision, linearity, range, LOQ |
| Impurities (limit) | Specificity, LOD |
| Assay / content | Specificity, accuracy, precision, linearity, range |

## Common pitfalls

- Treating **system suitability** as a substitute for validation — it confirms
  the system performs on the day, not that the method is valid.
- Skipping **robustness** evaluation, then being surprised by failures during
  transfer to QC.

> Tip: define acceptance criteria *before* you run the study, and justify them
> against the method's purpose — not against whatever the data happened to show.
